## :pushpin: Atividade 4:
4) Você está em uma sala com três interruptores, cada um conectado a uma lâmpada em uma sala diferente. Você não pode ver as lâmpadas da sala em que está, mas pode ligar e desligar os interruptores quantas vezes quiser. Seu objetivo é descobrir qual interruptor controla qual lâmpada.
Como você faria para descobrir, usando apenas duas idas até uma das salas das lâmpadas, qual interruptor controla cada lâmpada?

Resposta:

Ligo o primeiro interruptor, aguardo alguns minutos.

Depois disso, desligo o primeiro e ligo o segundo interruptor.

Vou até uma das salas. Se a lâmpada estiver acesa e fria, sabemos que o segundo interruptor a controla, se estiver acesa e quente, é o primeiro que a controla, se estiver apagada, é o terceiro.

Volto à sala, sabendo qual interruptor controla uma das lâmpadas.

Em seguida, ligo qualquer um dos dois restantes e dirijo-me a uma das salas que ainda não visitei, descobrindo qual interruptor a controla ao verificar se a lâmpada está acesa ou apagada.
